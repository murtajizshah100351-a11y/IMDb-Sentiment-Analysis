# Dataset instructions

This project uses the Kaggle dataset "IMDB Dataset of 50k Movie Reviews".
Download link: https://www.kaggle.com/datasets/lakshmi25npathi/imdb-dataset-of-50k-movie-reviews?resource=download

After download, place the CSV file here:
data/IMDB_Dataset.csv

If using Google Colab, you can mount your Drive and copy the file to drive/Mydrive and update the notebook paths.
